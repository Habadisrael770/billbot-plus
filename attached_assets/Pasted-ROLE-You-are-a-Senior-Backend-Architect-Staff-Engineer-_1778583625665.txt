PHASE 2 — JOB QUEUE SERVICE
Create:
artifacts/api-server/src/services/invoiceExtractionQueueService.ts

It must expose functions:
- enqueuePendingInvoices(limit?: number)
- enqueueInvoice(invoiceId: string, priority?: number)
- claimNextJobs(workerId: string, limit: number)
- markJobProcessing(jobId: string, workerId: string)
- markJobCompleted(jobId: string, invoiceId: string)
- markJobFailed(jobId: string, error: Error, retryable: boolean)
- markJobPoisoned(jobId: string, error: Error)
- getQueueStats()
- resetStaleProcessingJobs(staleMinutes: number)

Rules:
- claim only jobs with status queued/retrying and next_run_at <= now
- when claiming, set locked_by, locked_at, status=processing
- prevent two workers from claiming same job
- failed retryable jobs:
  - attempts += 1
  - if attempts < max_attempts → status=retrying, next_run_at = now + backoff
  - if attempts >= max_attempts → status=poisoned
- non-retryable jobs → failed or poisoned based on error type
- backoff:
  - attempt 1: 1 minute
  - attempt 2: 5 minutes
  - attempt 3: 15 minutes
- stale processing jobs older than 30 minutes return to retrying
