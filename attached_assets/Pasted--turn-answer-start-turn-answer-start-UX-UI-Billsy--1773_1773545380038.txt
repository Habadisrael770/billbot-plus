## פירוט מלא של המרכיבים

### 1️⃣ **תפריט עליון (Header)**

**אלמנטים:**
- **אייקון תפריט** (☰) - פותח את הניווט הצדדי
- **אייקון פרופיל** - לוגו Billsy עם טקסט "הגדרות"
- **חיפוש** - שורת חיפוש קלה לחיפוש מסמכים וספקים
- **אייקון התראות** (🔔) - הודעות ועדכונים
- **אייקון אפליקציה/מטבח** (⋮) - תפריט נוסף

**עיצוב:**
- רקע לבן/אפור בהיר
- טקסט בעברית מיישור מימין
- גובה סטנדרטי לתפריטים
- אייקונים בגודל קטן מובן ויהירותי

---

### 2️⃣ **אזור הפרופיל (Profile Section)**

**מרכיבים:**
- **אייקון פרופיל** - תמונה עגולה (avatar) עם סמל אדם
- **שם המשתמש** - "B1 pin" בגודל גדול (heading)
- **דוא"ל** - כתובת דוא"ל בטקסט קטן אפור

**עיצוב:**
- מרכוז ויזואלי בתחתית העמוד
- רווחים גדולים ליד הפרופיל
- טיפוגרפיה ברורה עם הבחנה בין שם לדוא"ל

---

### 3️⃣ **תפריט הגדרות (Settings Menu)**

**סדר פריטים (משמעל לימין):**

| אייקון | שם הסעיף | תיאור משנה | פעולה |
|--------|----------|-----------|-------|
| 🏢 | **פרטי עסק** | הגדרות חשבוניות ולוגו עסק | → |
| 👑 | **תוכנית ומנוי** | UNLIMITED / ניהול תשלומים | → |
| 👑 | **תוכניות** | בחירה או שינוי תוכנית | → |
| ⚡ | **שימוש ומגבלות** | שימוש החודש ומגבלות | → |
| 🔒 | **אבטחה ופרטיות** | שינוי סיסמה ואימות דו-שלבי | → |
| 📊 | **רואה חשבון** | ייצוא דוחות ואחסון קבצים | → |
| 🚪 | **יציאה** | התנתקות מהחשבון | - |

**עיצוב הפריטים:**
- **כל פריט** = בטון/ערימה שטוחה
- **סמל + כותרת** = בחלק שמאל
- **תיאור קטן** = בצבע אפור בהיר
- **חץ ימינה** = סמן ניווט (מלבד יציאה)
- **רווח בין פריטים** = ~12px
- **טקסט מצד ימין** (תמיכה RTL בעברית)

---

### 4️⃣ **תפריט תחתון (Bottom Navigation)**

**5 כפתורים:**

```
[לוח הבקרה] [חשבוניות] [+] [ספקים] [הגדרות✓]
```

**מרכיבים:**
- **לוח הבקרה** - אייקון גרף/ביתי
- **חשבוניות** - אייקון מסמכים
- **כפתור + מרכזי** - בצבע כהה (צבע ראשי טורקויז)
- **ספקים** - אייקון עמודות
- **הגדרות** - אייקון הגדרות עם ✓ (מסומן כפעיל)

**עיצוב:**
- **מוקד** = בתחתית המסך
- **צבע פעיל** = צבע טורקויז/ירוק כהה
- **צבע לא פעיל** = אפור
- **גובה** = ~60px עם padding
- **פונט קטן** = תווים בודדים למטה

---

### 5️⃣ **צבעים ופלטה**

| רכיב | צבע | קוד משוער |
|-----|------|----------|
| **צבע ראשי (Primary)** | טורקויז/ירוק כהה | `#138A8A` or `#117777` |
| **רקע** | לבן/אפור בהיר | `#FFFFFF` / `#F5F5F5` |
| **טקסט ראשי** | שחור כהה | `#333333` |
| **טקסט משנה** | אפור בינוני | `#999999` |
| **אייקונים** | תואם צבע ראשי | טורקויז |

---

### 6️⃣ **Typography**

| רכיב | גודל | משקל | צבע |
|-----|------|------|------|
| **שם המשתמש** | 18-20px | Bold | `#333333` |
| **תיאור משנה** | 13-14px | Regular | `#999999` |
| **כותרת פריט** | 15-16px | Medium | `#333333` |
| **תפריט תחתון** | 10-11px | Regular | אפור/טורקויז |

---

### 7️⃣ **ריווח (Spacing)**

- **Padding אזור ראשי** = 16-20px
- **Margin בין פריטים** = 12-16px
- **Padding בכפתורים** = 12px (top/bottom), 16px (sides)
- **גובה בטון** = 56-64px לכל פריט

---

### 8️⃣ **אינטראקציות**

- **Hover/Tap**: בטונים משנים את הצבע או רקע אפור בהיר
- **חץ ניווט** (→): מציין שהפריט ניתן ללחיצה
- **פעיל/לא פעיל**: הגדרות מסומנות בסימן ✓ וצבע ראשי

---

## קביעויות ה-UX/UI

✅ **אוהד טלפון ידידותי** - מבנה אנכי ברור וקל לגלילה
✅ **RTL (עברית)** - תמיכה מלאה בכתיבה מימין לשמאל
✅ **אייקונים אינטואיטיביים** - כל פריט בעל סמל בהיר
✅ **הבחנה חזותית** - טקסט ראשי וקטגוריות ברורות
✅ **Bottom Navigation** - אצבע קלה להשגה בטלפון
✅ **עקביות** - צבע ראשי אחד לכל האתר
