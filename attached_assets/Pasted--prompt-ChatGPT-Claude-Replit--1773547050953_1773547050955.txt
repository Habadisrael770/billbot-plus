## 📝 **PROMPT מדויק ל-Replit (HTML + CSS + JS)**

```
אתה מעצב UI/UX. צור עמוד HTML מרוסן עם CSS ו-JavaScript לתפריט נייד עברי (RTL).

דרישות:
1. תפריט hamburger שנפתח מימין (RTL)
2. כל קטגוריה בתפריט צריכה להיות בתוך מסגרת (CARD/BOX) עם border ורקע עדין
3. הנפשה חלקה:
   - תפריט נכנס עם slide-in מימין (300ms)
   - overlay שדהוק עם fade-in
   - כל קטגוריה מופיעה בעיכוב קטן (staggered animation - 50ms בין כל אחת)
   - כפתור X לסגירה עם rotate animation
4. אייקונים מ-Font Awesome (SVG inline אם אפשר)
5. צבעים:
   - רקע כלל: כחול כהה #0f1219
   - כפתור ראשי: כחול בהיר #4B7EF5
   - מסגרות: border כחול עדין עם background קצת בהיר
   - טקסט: לבן
6. על כל קטגוריה בתפריט:
   - אייקון משמאל (לפי סוג הקטגוריה)
   - טקסט יישור ימני
   - padding טוב
   - hover effect עם transition
7. קטגוריות בתפריט:
   - הצעה חדשה (כפתור כחול)
   - הצעות מחיר (icon: document)
   - סטטיסטיקות (icon: chart)
   - מסלולים (icon: path/route)
   - לקוחות (icon: users)
   - פריטים (icon: box)

צור קוד נקי, בנוי ויעיל עם תוכניות טובות.
```

✅ **אייקונים** - Font Awesome (6 אייקונים שונים)

✅ **Responsive** - עובד בנייד וביוקום
